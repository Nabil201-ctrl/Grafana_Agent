# Grafana Agent Configuration

This repository contains the configuration for a Grafana Agent.

## Files

*   `.gitignore`: This file specifies which files and directories to ignore in this Git repository.
*   `agent-config.alloy`: This is the main configuration file for the Grafana Agent, written in the Alloy configuration language.
*   `dockerfile`: This file contains the instructions for building a Docker image for the Grafana Agent with the provided configuration.
